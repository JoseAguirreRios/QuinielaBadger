QuinielaBadger

Plataforma de Quinielas Deportivas de Fútbol

¡Bienvenido a QuinielaBadger!

Esta plataforma te permite participar en quinielas deportivas y seguir los resultados de tus equipos favoritos de la Liga MX y otros torneos.

Existen dos formas de ejecutar esta aplicación. Por favor, selecciona la que mejor se adapte a tus necesidades:

Versión con Servidor Web

Recomendada para todas las funciones

Esta versión utiliza módulos ES6 y ofrece la mejor experiencia, pero requiere un servidor web local.

Requisitos:
  • Servidor web local (Node.js con http-server, Python, XAMPP, etc.)
  • Navegador moderno
Pasos:
  1. Instala Node.js desde nodejs.org
  2. Ejecuta npx http-server en la terminal desde la carpeta del proyecto
  3. O usa python -m http.server si tienes Python instalado
Usar Versión Completa

Versión Compatible

Para uso directo sin servidor

Esta versión funciona abriendo el archivo directamente, sin necesidad de un servidor web local.

Ventajas:
  • No requiere instalación de software adicional
  • Funciona abriendo directamente los archivos HTML
  • Compatible con más navegadores

Nota: Utiliza JavaScript tradicional sin módulos ES6 para mayor compatibilidad.
