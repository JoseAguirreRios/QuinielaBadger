QuinielaBadger - Versión Simple

¡Importante!

Esta es una versión simplificada de QuinielaBadger para solucionar problemas de compatibilidad al abrir archivos localmente.

Para usar todas las funcionalidades, te recomendamos utilizar uno de estos métodos:

  • Abre la aplicación a través de un servidor web local como http-server, Live Server, o XAMPP.
  • También puedes utilizar el índice normal (index.html) desde un servidor web.

Instrucciones para ejecutar con un servidor local:

  1. Instala Node.js desde nodejs.org
  2. Abre una terminal o línea de comandos en la carpeta del proyecto
  3. Instala http-server: npm install -g http-server
  4. Ejecuta el servidor: http-server
  5. Abre en tu navegador: http://localhost:8080
